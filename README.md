
## SIMPLIFIED SIMULATOR OF AMMONIA-WATER CYCLE

---



### Project Summary: System Modeling and Structured Simulation

This repository showcases the **"Most Simple" version** of a simulation for an ammonia-water absorption refrigeration cycle, developed in Python.

The project demonstrates the core engineering ability to translate physical principles (Energy and Mass Balance) into a functional, solvable code structure. The simulation successfully defines all cycle components (condenser, generator, evaporator, absorber, pump, and expansion valves), focusing on systematic and structured problem-solving.



### Application & Research Potential

This program serves as an foundational baseline for testing assumptions, integrating fault analysis, or serving as the base model for more complex optimization or Machine Learning applications.
